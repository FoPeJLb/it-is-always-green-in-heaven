Порядок установки:
##### pacman -S git
##### mkdir ~/gitclone
##### cd ~/gitclone
##### git clone https://aur.archlinux.org/pikaur.git
##### cd pikaur
##### makepkg -sri
##### pikaur -Syu
##### pikaur -S i3-gaps polybar i3-lock sakura
